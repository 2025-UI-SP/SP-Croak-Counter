## Running
1. Clone the repository
```bash
git clone git@github.com:2025-UI-SP/SP-Croak-Counter.git
```
2. Install dependencies
```bash
npm i
```
3. Run the program (development)
```bash
npm run dev
```
### Running With Docker (development only)
```bash
docker compose up
```
Open: http://localhost:5173

Stop container:
```bash
docker compose down
```
